About the Site

We condensed and redesigned the Seattle Schools website because we only used the original one for a couple links. We asked friends, teachers, and parents what they really used the SPS website for, and made those things quickly and easily accessable.

We're still adding features to this site. The next couple updates are going to be porting the urgent alerts, showing what the next days off are, and providing links to the nearest schools using location data. If you have suggestions, write to us below.
